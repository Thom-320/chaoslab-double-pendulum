0:00 - Hook

Mismas leyes, futuros distintos

Dos simulaciones obedecen la misma EDO. Solo cambia θ₂(0) en 10⁻⁶ rad.

perturbación microscópica efecto macroscópico
Abrir con contraste visual: al principio las trayectorias parecen una sola, luego se separan. La pregunta no es si las leyes fallan, sino por qué una diferencia tan pequeña puede amplificarse.

0:25 - Física I base

Primero, el caso ordenado

El ángulo se mide desde la vertical hacia abajo. Esa convención también se usa en el péndulo doble.

ω = dθ / dt

sin(θ) ≈ θ para |θ| ≪ 1

Esta escena ancla el proyecto en Física I: oscilaciones, gravedad, energía y rotación. El péndulo simple es el punto de partida familiar antes de mostrar el acoplamiento.

0:55 - Segunda masa

Agregar una barra cambia el juego

El estado pasa de una coordenada a cuatro variables acopladas.

s(t) = [θ1, ω1, θ2, ω2]

No cambiamos de leyes. Seguimos con mecánica clásica: gravedad, energía y rotación. Lo nuevo es que el movimiento de una barra afecta el de la otra.

1:20 - Modelo geométrico

De ángulos a posiciones

Las coordenadas salen por trigonometría, no por dibujo libre.

x1 = L1 sin(θ1)

y1 = -L1 cos(θ1)

x2 = x1 + L2 sin(θ2)

y2 = y1 - L2 cos(θ2)

Esta es la escena 3Blue1Brown: construir la posición paso a paso. Primero la masa 1 respecto al pivote; luego la masa 2 como desplazamiento relativo desde la masa 1.

1:55 - Simulación numérica

Resolver, no dibujar a mano

solve_ivp integra el sistema como problema de valor inicial.

d s / dt = f(t, s)

Cada frame sale del modelo físico. La animación no es una coreografía: es la visualización de una solución numérica reproducible.

2:20 - Espacio de ángulos

Mirar el movimiento como señal

En vez de seguir solo la masa, graficamos θ1 contra θ2.

orden local, geometría global compleja

Esta escena prepara el mapa global. Primero vemos una trayectoria individual en espacio de ángulos; luego repetimos el experimento para miles de condiciones iniciales.

2:45 - Control de calidad

La energía total casi no se mueve

Si el integrador inventara energía, no podríamos confiar en la sensibilidad observada.

E(t) = T(t) + V(t) ≈ constante

deriva relativa máx. ≈ 2.6×10⁻⁶
La energía total es el ancla de confianza. La divergencia no se debe a que el modelo viole groseramente la conservación de energía.

3:20 - Sensibilidad

Δ(t) crece

Medimos la separación entre dos estados casi idénticos en escala logarítmica.

Δ(t) = ||s(t) - s'(t)||

No es un exponente de Lyapunov formal: es evidencia numérica de sensibilidad.

Decir explícitamente que no estamos vendiendo una prueba formal de caos. Mostramos una medición de divergencia temprana compatible con sensibilidad a condiciones iniciales.

3:55 - Mapa global

Un mapa de futuros

Cada pixel nace de resolver la misma EDO con una condición inicial distinta.

Color = tiempo hasta el primer flip.

Zonas oscuras = no flip en la ventana simulada.

Este es el momento inspirado en 2swap/Observable. No presentar el mapa como imagen pegada: cada punto es un experimento computacional. La frontera irregular es el mensaje.

4:35 - Cierre

Determinismo no significa predictibilidad práctica

La física da las reglas. La computación revela sus consecuencias.

Física I + Cálculo I + EDOs + visualización científica

Alcance honesto: sin rozamiento y sin Lyapunov formal; sí hay simulación reproducible, energía controlada, divergencia y mapa de condiciones iniciales.

Cerrar con la idea central y las limitaciones. Eso protege la defensa oral: no prometemos una prueba matemática completa; defendemos evidencia numérica clara.
