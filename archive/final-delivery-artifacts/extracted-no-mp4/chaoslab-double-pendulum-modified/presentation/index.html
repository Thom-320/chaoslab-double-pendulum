0:00 - Hook

Mismas leyes, futuros distintos

Una diferencia inicial de 1e-6 rad puede volverse macroscopica.

Dos sistemas casi iguales se separan. La pregunta no es si las leyes fallan, sino por que amplifican diferencias pequenas.

0:25 - Fisica I base

Primero, el caso ordenado

El angulo se mide desde la vertical hacia abajo. Esa convencion tambien se usa en el pendulo doble.

ω = dθ / dt

sin(θ) ≈ θ para |θ| ≪ 1

Esta escena ancla el proyecto en Fisica I y Calculo I: derivadas, oscilacion y aproximacion de angulo pequeno.

0:55 - Segunda masa

Agregar una barra cambia el juego

El estado pasa de una coordenada a cuatro variables acopladas.

s(t) = [θ1, ω1, θ2, ω2]

No cambiamos de leyes. Seguimos con gravedad, energia y rotacion, pero ahora los grados de libertad se acoplan.

1:20 - Modelo geometrico

De angulos a posiciones

Las coordenadas salen por trigonometria, no por dibujo libre.

x1 = L1 sin(θ1)

y1 = -L1 cos(θ1)

x2 = x1 + L2 sin(θ2)

y2 = y1 - L2 cos(θ2)

Esta es la escena puente: primero construimos la posicion de la primera masa, luego la segunda como desplazamiento relativo desde la primera.

1:55 - Simulacion numerica

Resolver, no dibujar a mano

solve_ivp integra el sistema como problema de valor inicial.

d s / dt = f(t, s)

Cada frame sale del modelo fisico. La animacion es visualizacion de una solucion numerica.

2:30 - Control de calidad

La energia total casi no se mueve

Si el integrador inventara energia, no podriamos confiar en el caos observado.

E(t) = T(t) + V(t) ≈ constante

La energia total es el ancla de confianza. La divergencia no se debe a que el modelo viole la conservacion de energia.

3:05 - Sensibilidad

Δ(t) crece

Medimos la separacion entre dos estados casi identicos en escala logaritmica.

Δ(t) = ||s(t) - s'(t)||

No lo llamamos exponente de Lyapunov formal. Es una evidencia cuantitativa de sensibilidad a condiciones iniciales.

3:45 - Observable finale

Un mapa de futuros

Color = tiempo hasta el primer flip. Las fronteras revelan sensibilidad.

Este es el momento 2swap: una regla determinista local produce una imagen global compleja.

4:05 - Islas de estabilidad

Islas de orden entre el caos

Las regiones oscuras albergan trayectorias repetitivas como el pretzel; las zonas de color son caóticas.

Región negra ⇒ pendulo coherente; frontera ⇒ alta sensibilidad

Después de presentar el mapa global, esta escena señala que no todo es caos: existen 'islas' donde el sistema sigue un patrón periódico (pretzel, shoelace, corazón). Estas zonas muestran que la energía baja puede producir órbitas regulares incluso en un sistema no lineal.

4:35 - Cierre

Deterministico no significa predecible

La fisica da las reglas. La computacion revela sus consecuencias.

Fisica I + Calculo I + EDOs + visualizacion cientifica

Cerrar con limitaciones: sin rozamiento, mapa exploratorio, validacion por video como trabajo futuro.
