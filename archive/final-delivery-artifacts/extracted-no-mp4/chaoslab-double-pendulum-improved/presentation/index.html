0:00 - Hook

Mismas leyes, futuros distintos

Una diferencia inicial de 1e-6 rad puede volverse macroscópica.

Dos sistemas casi iguales se separan. La pregunta no es si las leyes fallan, sino por qué amplifican diferencias pequeñas.

0:25 - Física I base

Primero, el caso ordenado

El ángulo se mide desde la vertical hacia abajo. Esa convención también se usa en el péndulo doble.

ω = dθ / dt

sin(θ) ≈ θ para |θ| ≪ 1

Esta escena ancla el proyecto en Física I y Cálculo I: derivadas, oscilación y aproximación de ángulo pequeño.

0:55 - Segunda masa

Agregar una barra cambia el juego

El estado pasa de una coordenada a cuatro variables acopladas.

s(t) = [θ1, ω1, θ2, ω2]

No cambiamos de leyes. Seguimos con gravedad, energía y rotación, pero ahora los grados de libertad se acoplan.

1:20 - Modelo geométrico

De ángulos a posiciones

Las coordenadas salen por trigonometría, no por dibujo libre.

x1 = L1 sin(θ1)

y1 = -L1 cos(θ1)

x2 = x1 + L2 sin(θ2)

y2 = y1 - L2 cos(θ2)

Esta es la escena puente: primero construimos la posición de la primera masa, luego la segunda como desplazamiento relativo desde la primera.

1:55 - Simulación numérica

Resolver, no dibujar a mano

solve_ivp integra el sistema como problema de valor inicial.

d s / dt = f(t, s)

Cada frame sale del modelo físico. La animación es visualización de una solución numérica.

2:20 - Espacio de ángulos

Mirar el movimiento como señal

En vez de seguir solo la masa, graficamos θ1 contra θ2.

orden local, geometría global compleja

Esta escena prepara el mapa global. Primero vemos una trayectoria individual en espacio de ángulos; luego repetimos el experimento para miles de condiciones iniciales.

2:30 - Control de calidad

La energía total casi no se mueve

Si el integrador inventara energía, no podríamos confiar en el caos observado.

E(t) = T(t) + V(t) ≈ constante

deriva relativa máxima ≈ 2.56×10-6

La energía total es el ancla de confianza. La divergencia no se debe a que el modelo viole la conservación de energía.

3:05 - Sensibilidad

Δ(t) crece

Medimos la separación entre dos estados casi idénticos en escala logarítmica.

Δ(t) = ||s(t) - s'(t)||

pendiente temprana de log Δ ≈ 1.10 s-1

No lo llamamos exponente de Lyapunov formal. Es evidencia numérica de sensibilidad a condiciones iniciales.
