# Guion estilo 2swap / 3Blue1Brown para video corto

## Tono visual
Fondo oscuro, líneas finas, color neón, transición suave entre fórmula, simulación y mapa. Nada de diapositivas llenas de texto. La idea es que el espectador vea primero el fenómeno y después entienda la ecuación. Sí, una decisión pedagógica sensata; no avisemos demasiado para no asustar.

## Estructura de 90 segundos

### 0:00–0:10 — Gancho
Pantalla negra. Aparecen dos péndulos casi idénticos.
Texto: “Dos sistemas. Misma física. Diferencia inicial: 0.000001 rad.”
Ambos empiezan sincronizados.

### 0:10–0:25 — Ruptura
Las trayectorias se separan. Aparece la curva Delta(t) creciendo.
Narración: “El problema no es que no conozcamos las leyes. El problema es que las leyes amplifican errores microscópicos.”

### 0:25–0:45 — Física
Aparecen variables theta1, theta2, omega1, omega2.
Se muestra E = T + V casi constante.
Narración: “La energía se conserva, así que el caos no viene de un error numérico. Viene de la no linealidad.”

### 0:45–1:05 — Mapa fractal
Aparece el mapa de condiciones iniciales. Cada pixel es un experimento.
Texto: “Color = tiempo hasta el primer flip.”
Narración: “Cambiar apenas el ángulo inicial puede cambiar completamente el futuro del sistema.”

### 1:05–1:20 — Conclusión
Volver al péndulo simple y luego al doble.
Narración: “El péndulo simple enseña periodicidad. El péndulo doble enseña humildad.”

### 1:20–1:30 — Cierre
Título del proyecto, autores, curso, GitHub.
